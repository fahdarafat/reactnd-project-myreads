# MyReads Project
This project was created for the React-cross-skilling Nanodegree for Udacity.
## Install Dependencies 
`npm install`

## Run The project locally
`npm run start`

This will run the app on http://localhost:3000/

## How to use the app

 - The main page is separated in to three shelves where you can track books by the following categories (Want to read, Currently reading, Read)

- You can change the state of each book by clicking on the green dropdown arrow. 
- You can navigate to the search page http://localhost:3000/search and search for more books using the search terms mentioned below.
